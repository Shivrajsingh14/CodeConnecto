### ğŸ› ï¸ Quick Start Installation

**1. Repository Setup**
```bash
# Clone the repository with all branches and tags
git clone --recurse-submodules https://github.com/Shivrajsingh14/CodeConnecto.git
cd CodeConnecto

# Install development tools globally (optional but recommended)
npm install -g pnpm typescript nodemon pm2 @types/node
```

**2. Environment Configuration**
```bash
# Copy environment templates
cp client/.env.example client/.env
cp server/.env.example server/.env

# Generate secure JWT secret
node -e "console.log(require('crypto').randomBytes(64).toString('hex'))"
```

**Frontend Configuration (`client/.env`)**
```env
# Development Configuration
VITE_BACKEND_URL=http://localhost:3000
VITE_WS_URL=ws://localhost:3000
VITE_AI_API_KEY=your_ai_api_key
VITE_DAILY_API_KEY=your_daily_co_api_key
VITE_ENV=development

# Optional: Analytics & Monitoring
VITE_ANALYTICS_ID=your_analytics_id
VITE_SENTRY_DSN=your_sentry_dsn
```

**Backend Configuration (`server/.env`)**
```env
# Server Configuration
NODE_ENV=development
PORT=3000
HOST=localhost

# Database Configuration  
MONGO_URI=mongodb://localhost:27017/codeconnecto
REDIS_URL=redis://localhost:6379

# Authentication & Security
JWT_SECRET=your_generated_jwt_secret_here
JWT_EXPIRES_IN=7d
BCRYPT_ROUNDS=12

# Email Service (SMTP)
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password

# AI Integration
GOOGLE_AI_API_KEY=your_google_ai_key
AI_MODEL=gemini-pro

# Payment Gateways
RAZORPAY_KEY_ID=your_razorpay_key
RAZORPAY_SECRET=your_razorpay_secret
STRIPE_SECRET_KEY=your_stripe_secret

# WebRTC Configuration
DAILY_API_KEY=your_daily_co_api_key
STUN_SERVER=stun:stun.l.google.com:19302
```

**3. Dependency Installation & Build**
```bash
# Install all dependencies (root + client + server)
npm install

# Install client dependencies
cd client && npm install

# Install server dependencies  
cd ../server && npm install

# Build TypeScript files
npm run build
```

**4. Database Setup**
```bash
# Start MongoDB (if running locally)
sudo systemctl start mongod

# Start Redis (if running locally)  
sudo systemctl start redis-server

# Initialize database with sample data (optional)
cd server && npm run db:seed
```

**5. Development Servers**
```bash
# Terminal 1: Start backend server with hot reload
cd server && npm run dev

# Terminal 2: Start frontend development server
cd client && npm run dev

# Terminal 3: Start Redis and MongoDB (if local)
redis-server & mongod

# Access application at http://localhost:5173
```
